Starter code¶

Retrieving all the data from online and importing into a panda dataframe. Also, seperating the location string into two seperate columns.

The Location 1 column is a string of the format (float,float). Using simple string manipulation, we take it apart and create a latitude and longitude column for each row

import folium
import requests
import pandas

arrest_table = pandas.read_csv("http://www.hcbravo.org/IntroDataSci/misc/BPD_Arrests.csv")

arrest_table["race_new"] = arrest_table["sex"]
arrest_table["sex_new"] = arrest_table["race"]
arrest_table["race"] = arrest_table["sex_new"]
arrest_table["sex"] = arrest_table["race_new"]
arrest_table = arrest_table.drop('race_new', 1)
arrest_table = arrest_table.drop('sex_new', 1)

arrest_table = arrest_table[pandas.notnull(arrest_table["Location 1"])]

arrest_table["lat"], arrest_table["long"] = arrest_table["Location 1"].str.split(",").str
arrest_table["lat"] = arrest_table["lat"].str.replace("(", "").astype(float)
arrest_table["long"] = arrest_table["long"].str.replace(")", "").astype(float)

My Code - Basic data processing¶

Begin processing all the data and adding pop ups onto the map. Each pop up is added using the location data (lat,long) of each arrest. The pop up contains relevant information such as the date, address, and incident offense for that particular arrest.

For this particular example, I set the number of pop ups generated to be 100, however, one could easily modify this limit by changing the while loop parameters.

map_osm = folium.Map(location=[39.29, -76.61], zoom_start=11)
map_osm

i = 0
nummarkers = 0

while nummarkers < 100:
    tempx = arrest_table.at[i,'lat']
    tempy = arrest_table.at[i,'long']
    #folium.Marker([tempx, tempy], popup='Hi <br> lol').add_to(map_osm)
    currIncident = arrest_table.at[i,'incidentOffense']
    currAddress = arrest_table.at[i,'arrestLocation']
    currDate = arrest_table.at[i,'arrestDate']
    currAge = str(arrest_table.at[i,'age'].item())
    currRace = arrest_table.at[i,'race']
    currSex = arrest_table.at[i,'sex']
    info = ('<b>Address</b> : ' + currAddress + '<br> <b>Date</b> : ' + currDate + '<br> <br> <b>Incident</b> : ' + currIncident +
    '<br> <b>Age<b/> : ' + currAge + '<br> <b>Race<b/> : ' + currRace + '<br> <b>Sex</b> : ' + currSex)
    
    if (currIncident == '87-Narcotics' ):
        folium.Marker(
        location=[tempx, tempy],
        popup=info,
        icon=folium.Icon(color='green')
        ).add_to(map_osm)
        nummarkers += 1
    elif ( currIncident == '4E-Common Assault'):
        folium.Marker(
        location=[tempx, tempy],
        popup=info,
        icon=folium.Icon(color='darkred')
        ).add_to(map_osm)
        nummarkers += 1
    elif ( currIncident == '54-Armed Person'):
        folium.Marker(
        location=[tempx, tempy],
        popup=info,
        icon=folium.Icon(color='red')
        ).add_to(map_osm)
        nummarkers += 1
    elif ( currIncident == '4B-Agg. Asslt.- Cut'):
        folium.Marker(
        location=[tempx, tempy],
        popup=info,
        icon=folium.Icon(color='orange')
        ).add_to(map_osm)
        nummarkers += 1
    elif ( currIncident == '55-Disorderly Person'):
        folium.Marker(
        location=[tempx, tempy],
        popup=info,
        icon=folium.Icon(color='blue')
        ).add_to(map_osm)
        nummarkers += 1
    
    i = i + 1
